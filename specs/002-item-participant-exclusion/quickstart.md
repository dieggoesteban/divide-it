# Quickstart

## Prerequisites
- Node.js 18+
- npm or yarn

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

## Running Development Server
1. Start Vite server:
   ```bash
   npm run dev
   ```
2. Open http://localhost:5173

## Testing
- Manual testing via UI.
- Verify calculations by adding participants with specific exclusions.
