# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

## [1.1.0] - 2024-12-20

### Added
- Itemized expense breakdown: Participants can add multiple items with descriptions and amounts instead of just a total
- Participant exclusion per item: Exclude specific participants from individual items for fairer expense splitting

### Changed
- Refactored expense calculation logic to compute debt per item instead of globally

## [1.0.0] - 2024-12-18

### Added
- Initial project setup with React, TypeScript, and Vite.
- Integrated shadcn/ui components.
- Established project constitution and governance.
- Basic expense splitting functionality with pnparticipant management
- Balance calculation and debt settlement suggestions
- Responsive UI with Tailwind CSS
